jQuery中remove()函数的使用

在前端编程时，我们又是需要使用类似parent()、children()、prev()、next()之类的方法遍历DOM树来获取其他元素的信息， 而先对起始节点所在的分支做remove()操作会使以上方法失效。


比如：

    function removeimg(thislabel) {
        $(thislabel).remove();
        $(thislabel).parent().next().show();
        
    }

这样第二条语句是会报错的，如果使用alert($(thislabel).parent().html())，会显示undefined


正确的顺序应该是先操作、后移除：

    function removeimg(thislabel) {
        $(thislabel).parent().next().show();
        $(thislabel).remove();
    }
